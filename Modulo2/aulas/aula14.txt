# Estrutura "while"
    Estrutura de repetição com teste lógico.

Algorítmo:

enquanto não (maçã)
    se terra
        passo
    se vazio
        pula
    se moeda
        pega
pega

Exemplo:

while not maçã:
    if chão:
        passo
    if buraco:
        pula
    if moeda:
        pega
pega
